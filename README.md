# Insignia-Subnet
The Insignia subnet utilizes the bittensor network to structure a decentralized quantitative trading firm inspired by how a standard quant shop operates. Layer 1 miners compete to produce predictive machine learning models, scored by validators against a proprietary tick-by-tick benchmark dataset. Winning models from Layer 1 are not deployed directly. Instead, they are passed to Layer 2 miners who build and execute live trading strategies inferencing those models. Layer 2 validator performance is measured against real time performance, producing battle-tested models and strategies eligible for deployment by the subnet owner’s prop trading firm.
Multiple strategies are used to economically support inscentives and investors. The firm uses profits from the top deployed models for alpha token buy backs in order to counter sell pressure from miners. API access to the firm's top real time strategies are accessable to top holders of the alpha token, suppessing sell pressure from whales. Any external investments from quant shops or investors amplifies inflows and inscentives.
