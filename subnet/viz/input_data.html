Feature Importance — What the model relies on most

Top 20 Features by Permutation Importance

Measured by accuracy drop when each feature is shuffled. The sections below are ordered by this ranking.
#1 Spot Flow / CVD (23 features) — Top feature: spot_cvd_agg

Spot CVD (Cumulative Volume Delta)

Running total of (buy volume - sell volume) on spot. Rising = net buying. Divergence from price = potential reversal.

Spot CVD Z-Scores

How extreme the current CVD is relative to recent history. Values beyond ±2 = unusual flow.
#2 Open Interest (31 features) — Top features: oi_low, oi_coin_high, oi_high

Open Interest (USD)

Total value of outstanding futures contracts. Rising OI + rising price = strong trend. Rising OI + falling price = bearish pressure.

OI Change (multi-timeframe)

Rate of change in open interest over 30min, 1h, 4h windows.
#3 Order Book (21 features) — Top feature: fut_ob_imbalance_ma24

Futures Order Book Imbalance

Bid vs ask depth imbalance. Positive = more bids (buy pressure). Negative = more asks (sell pressure).

CoinGlass Depth Ratio

Aggregated bid/ask depth ratio across exchanges. Above 1.0 = more buy-side liquidity.
#4 Funding Rate (18 features) — Top feature: funding_change_4h

Funding Rate

Cost of holding leveraged long positions. Positive = longs pay shorts (bullish crowding). Negative = shorts pay longs.
#5 Taker Flow (12 features)

Taker Buy vs Sell (Futures)

Aggressive market orders. Taker buys = someone paying the ask (bullish urgency). Taker sells = hitting the bid.
#6 Liquidations (6 features)

Liquidation Volume (Long vs Short)

Forced position closures. Long liquidations = longs getting stopped out (bearish cascade). Short liquidations = short squeeze.
